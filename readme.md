 # Modern Calculator App



A sleek, functional, and modern web-based calculator. This project was developed by me to offer a clean and intuitive user interface inspired by contemporary calculator designs, along with essential arithmetic capabilities.



 ## Features



 -    * *Clean and Intuitive UI: * * User-friendly and visually appealing.

 -    * *Basic Arithmetic: * * Supports addition, subtraction, multiplication, and percentage.

 -    * *Responsive Design: * * Adapts to various screen sizes.

 -    * *Clear Display: * * Shows both current expression and result.

 -    * *Error Handling: * * Basic error management for calculations.



 ## Technologies



 -    * *HTML5 * *

 -    * *CSS3 * *

 -    * *JavaScript (ES6+) * *

 ## Getting Started



 ### Prerequisites



A modern web browser.



 ### Installation



1 .  Clone this repository:

    ```bash

    git clone https://github.com/phi347/modern-calculator.git

    ```

2 .  Navigate to the directory:

    ```bash

    cd modern-calculator

    ```

3 .  Open `index.html` in your browser.



 ## Usage



 -   Input numbers and operators by clicking the respective buttons.

 -   `C` clears the display.

 -   `+/-` toggles the sign.

 -   `=` calculates the result.
